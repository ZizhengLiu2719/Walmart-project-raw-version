# Transport Service - General Info
Creator: Michael
Data Format: CSV

## Dependencies
- Java 21
- Maven 3+
- Spring Boot 3.3.3
- OpenCSV 5.9

## How to Run
1. Navigate to the `DataProviders/Transport_Service` directory.
2. Build the project using Maven:
   ```bash
   mvn package
   ```
3. Run the compiled JAR file:
   ```bash
   java -jar target/transport-service-0.0.1-SNAPSHOT.jar
   ```
The service will start on port 8083.
